# Project J.A.R.V.I.S.
### Just A Rather Very Intelligent System

Modular autonomous system integrating high-systematics trading (SMC/ICT) with robotics engineering. Core architecture for automated market analysis, embodied AI, and mechanical prototyping.

## 🚀 Core Architecture

* **Omni-Node (Finance):** A high-systematics trading engine focused on Smart Money Concepts (SMC) and ICT methodologies. It utilizes 1-hour and 4-hour timeframe analysis.
* **Neuro-Link (AI):** The cognitive layer. Currently being developed to handle system-wide command execution via mobile terminal environments (Termux/Pydroid 3).
* **Mech-Core (Robotics):** The hardware abstraction layer. Focused on kinematics for quadrupedal robotics and eventual exoskeleton integration.

## 📈 Roadmap

- [ ] **Phase 1: Foundation** - Finalizing the NewsChamber filtering module and 4-hour liquidity scanner.
- [ ] **Phase 2: Embodied AI** - Integration of Python-based kinematics for robotic limb control.
- [ ] **Phase 3: Physical Prototyping** - Transitioning to hardware assembly in Shenzhen.
